## 8X8X8 LED CUBE  
  
  
-**프로젝트 기간** : 2011/09~ 2011/11  

-**소속** : 학교 프로젝트  

-**주사용 기술** : ATmega C 코딩 담당 및 HW 제작
  
  
-**프로젝트 내용**  
  
학교 2인 팀 프로젝트에서 ATmega128을 이용하여 8X8X8 LED CUBE를 제작하였다.
LED의 0,1의 동작 로직을 이용하여 3차원 배열 애니메이션을 제작한 작품이다.
현재는 관리 미흡으로 소스는 분실된 상태이다.
